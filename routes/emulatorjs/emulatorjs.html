Instructions

Keyboard Controls

  • Arrow Keys ↑↓←→ - Move
  • A Button Z
  • B Button X
  • X Button A
  • Y Button S
  • Start Enter
  • Select Shift

Gamepad Support

  • USB gamepads are automatically detected
  • Standard mapping is supported
  • Connect gamepad before loading game
  • Click Settings button, then select "Gamepad" to configure buttons
  • Click game screen to get focus
  • Press any gamepad button to activate

Save & Load

  • F5 Quick Save
  • F7 Quick Load
  • Use emulator settings menu for more options
  • Saves stored in browser

Fullscreen

  • Click "Fullscreen" button to enter
  • Press ESC to exit fullscreen
